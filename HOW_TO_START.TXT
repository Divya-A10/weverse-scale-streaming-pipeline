Step 3: Create Your Database and User
By default, PostgreSQL creates a default superuser named after your Mac's login username (which is divyaavanigadda). Let's log into the system console to create a dedicated database for your project.

Open the PostgreSQL console:

Bash
psql postgres
(Your terminal prompt will change to postgres=#, meaning you are inside the database).

Create a brand new database named weverse_analytics:

SQL
CREATE DATABASE weverse_analytics;
Connect to your new database:

SQL
\c weverse_analytics;
